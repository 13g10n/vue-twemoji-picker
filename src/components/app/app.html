Cool Emoji Picker
Cool Emoji Picker was created to be a developer-friendly library for anyone who wants to add emojis support in their Vue app using the Twemoji (Twitter) arts. It comes with two components:
CoolPicker and CoolTextArea.
CoolPicker is meant to be a plug-n-play emoji picker only. It emit some events like @emojiUnicodeAdded (check component docs for more info) so you can catch the selected emoji selected and perform some action.
CoolTextArea is meant to be a plug-n-play emoji picker + textarea which provides string sanitization for HTML tags and emoji support.

Cool Emoji Picker emoji picker uses Emoji Unicode 11 (latest emoji spec, from 2018, which includes skin tones) as its base, and apply some trimming on the emoji list for better optimization of component size. Also, for the sake of your bundle size, you can opt if you want all emoji groups, or only smileys, or even only smileys and foods! The choice is yours!

Requirements:
- Cool Emoji Picker is made with Vue, for Vue (for the time being).
- For CoolPicker component search feature to work your browser needs to support String Includes.
- For CoolTextArea component layouting proportions correctly render your browser needs to support CSS Flexbox.
- In most cases Polyfill.io can ensure support for older browsers.
Any suggestion or bug please report at Issues
Getting Started
- To install Cool Emoji Picker on your Vue project you can run, for NPM:
npm install cool-emoji-picker
- For Yarn:
yarn add cool-emoji-picker
- To import CoolPicker on your Vue project you can do as follows:
JAVASCRIPT: import { CoolPicker } from 'cool-emoji-picker'; import EmojiData from 'cool-emoji-picker/src/emoji-data/emoji-all-groups.json'; import EmojiGroups from 'cool-emoji-picker/src/emoji-data/emoji-groups.json'; export default { name: 'App', components: { 'coolpicker': CoolPicker }, computed: { emojiDataAll() { return EmojiData; }, emojiGroups() { return EmojiGroups; } } } TEMPLATE: <coolpicker :emojiData="emojiDataAll" :emojiGroups="emojiGroups" :skinToneSelection="true" @emojiUnicodeAdded="yourMethodToCatchEmojiUnicode" @emojiImgAdded="yourMethodToCatchEmojiImg" ></coolpicker>
- To import CoolTextArea on your Vue project you can do as follows:
JAVASCRIPT: import { CoolTextArea } from 'cool-emoji-picker'; import EmojiData from 'cool-emoji-picker/src/emoji-data/emoji-all-groups.json'; import EmojiGroups from 'cool-emoji-picker/src/emoji-data/emoji-groups.json'; export default { name: 'App', components: { 'cooltextarea': CoolTextarea }, computed: { emojiDataAll() { return EmojiData; }, emojiGroups() { return EmojiGroups; } } } TEMPLATE: <cooltextarea :content.sync="content" :emojiData="emojiDataAll" :emojiGroups="emojiGroups" @enterKey="onEnterKey" ref="cooltextareaRef"></cooltextarea>
These are basic examples for importing Cool Emoji Picker components in a Vue project.
CoolPicker Docs
CoolPicker - Component Attributes
Attribute Type Valid Values Description
pickerWidth Number Any Number Define the emoji picker width (in pixels). Default: '250'.
pickerMaxHeight Number Any Number Define the emoji picker max height (in pixels). Default: '200'.
emojiData Array Array with the Specified Structure This is the array of emojis which is supplied by this library, like exemplified in Emoji Data Index. Default: [].
emojiGroups Array Array with the Specified Structure This is the array of emoji groups which is supplied by this library, like exemplified in Emoji Data Index. Default: [].
skinToneSelection Boolean true / false This is a boolean to enable or disable skin tone selection in the emoji picker. Default: false.
recentEmojisFeat Boolean true / false This is a boolean to enable or disable recent emojis feature in the emoji picker. Default: false.
recentEmojisStorage String 'local', 'session', 'none' Sets the type of storage for recent emojis in a browser. 'local': LocalStorage; 'session': SessionStorage; 'none': Disable storage. Default: 'none'.
recentEmojiStorageName String Any String Sets the name of the browser storage object, if enabled. Default: 'cep-recent-emojis'.
recentEmojiLimit Number Any Number Sets the maximum amount of recent emojis. Default: 12.
searchEmojisFeat Boolean true / false This is a boolean to enable or disable search emojis feature in the emoji picker. Default: false.
searchEmojiPlaceholder String Any String Define the search input placeholder. Default: 'Search emojis.'.
searchEmojiNotFound String Any String Define the label when there are no emojis found. Default: 'No emojis found.'.
twemojiPath String Any String URL This is a variable to define where are stored the emoji arts static resources. Default: 'https://twemoji.maxcdn.com/2/'.
twemojiExtension String '.png', '.svg', '.jpg', '.jpeg', '.ico' This is a variable to define emoji arts static resources extension. Note: This property only changes the extension, the filename will always be the codepoint for the emoji Unicode. Default: '.png'.
twemojiFolder String Any String folder name This is a variable to define the folder where are stored the emoji arts static resources in the 'twemojiPath'. Default: '72x72'.
btnEmojiClasses Array of Strings Array of Strings (CSS classes) This is an array of CSS classes for those who wish to further customize the style of the emoji button of the component. Default: [].
CoolPicker - Component Events
Event Parameter Description
emojiUnicodeAdded emojiUnicode This event fires everytime the user selects an emoji in the picker, and its parameter is the emoji in its Unicode format.
emojiImgAdded emojiImgTag This event fires everytime the user selects an emoji in the picker, and its parameter is the emoji HTML IMG tag.

CoolTextarea - Useful Methods
These are useful methods of the component you can access if you define a ref attribute to your component. The method can then be accessed on your Vue parent component with:
this.$refs.yourRefName.methodName
Method Parameter Description
addText(text) text - String This method adds a text to the textarea. The text parameter will then be sanitized and emojis Unicodes converted to emoji IMG arts from Twemoji.
addTextBlur(text) text - String This method adds a text to the textarea. The text parameter will then be sanitized and emojis Unicodes converted to emoji IMG arts from Twemoji. It is the same as addText(text) method, but this one doesn't trigger smartphone keyboard.
cleanText() None This method cleans the textarea.
setSkinTone(tone) tone - Number (a number >= 0 and <= 5) This method sets the skin tone active in the emoji picker.
Emoji Data Index
As seen in the example you will need, for the emoji picker to work, to import an indexed list of emojis and emoji groups name (which is provided by this lib) and pass as a prop of the CoolTextArea/CoolPicker component. There is already different bundles so you are able to choose if you want your emoji picker to have all emojis or only smileys, or only smileys and objects, etc.
This library execute some optimizations excluding unnecessary fields (for this component) in an already indexed list of Unicode emojis from the lib 'emojibase'. Below you have info for each emoji bundle:

PS: You can provide your own emoji.json index file with a list of emoji Unicodes, but you will need to follow the structure of the bundles generated by this library (like cool-emoji-picker/src/emoji-data/emoji-all-groups.json).
PS: You can provide your own emoji-groups.json index file with a list of emoji groups name, but you will need to follow the structure of the bundle generated by this library (cool-emoji-picker/src/emoji-data/emoji-groups.json).
