CoolTextArea
A plug-n-play [Tw]emoji Picker and Textarea for Vue
CoolTextArea is a modern textarea component built with Vue which includes a emoji picker (and supports latest emoji Unicode specs, including skin tone selection).

CoolTextArea is meant to be a plug-n-play textarea which provides string sanitization for HTML tags and emoji support.

CoolTextArea emoji picker uses Emoji Unicode 11 (latest emoji spec, from 2018, which includes skin tones) as its base, and apply some trimming on the emoji list for better optimization of component size. Also, for the sake of your bundle size, you can opt if you want all emoji groups, or only smileys, or even only smileys and foods! The choice is yours!

Requirements:
- CoolTextArea is made with Vue, for Vue (for the time being).
- For the emoji picker to work your browser needs to support IntersectionObserver.
- For CoolTextArea layouting proportions correctly render your browser needs to support CSS Flexbox.
Any suggestion or bug please report at Issues
Getting Started
- To install CoolTextArea on your Vue project you can run, for NPM:
npm install cooltextarea
- For Yarn:
yarn add cooltextarea
- To import CoolTextArea on your Vue project you can do as follows:
JAVASCRIPT: import CoolTextArea from 'cooltextarea'; import EmojiData from 'cooltextarea/src/emoji-data/emoji-all-groups.json'; import EmojiGroups from 'cooltextarea/src/emoji-data/emoji-groups.json'; export default { name: 'App', components: { 'cooltextarea': CoolTextarea }, computed: { emojiDataAll() { return EmojiData; }, emojiGroups() { return EmojiGroups; } } } TEMPLATE: <cooltextarea :content.sync="content" :emojiData="emojiDataAll" :emojiGroups="emojiGroups" @enterKey="onEnterKey" ref="cooltextareaRef"></cooltextarea>
This is a basic example for importing CoolTextArea in a Vue project. You will, of course, want to choose the emoji collections you want to import, if you want to display the emoji picker only, or the textarea only, or don't display the send button.
Detailed documentation are on sections Attributes, Events, and Emoji Data Index below.
Component Attributes
Attribute Type Valid Values Description
content String Any String Variable This is the content variable which will be the output of the textarea with the emoji art IMG tags converted to its correspondent Unicode.
emojiData Array Array with the Specified Structure This is the array of emojis which is supplied by this library, like exemplified in Emoji Data Index. Default: [].
emojiGroups Array Array with the Specified Structure This is the array of emoji groups which is supplied by this library, like exemplified in Emoji Data Index. Default: [].
skinToneSelection Boolean True / False This is a boolean to enable or disable skin tone selection in the emoji picker. Default: false.
enableEmojiPicker Boolean True / False This is a boolean to enable or disable the emoji picker. Default: true.
enableTextarea Boolean True / False This is a boolean to enable or disable the textarea. Default: true.
enableSendBtn Boolean True / False This is a boolean to enable or disable the send button. Default: false.
componentColor String 'cream', 'cherry', 'forest', 'ocean', 'sun' This is a variable to define the component color. Default: 'cream'.
twemojiPath String Any String URL This is a variable to define where are stored the emoji arts static resources. Default: 'https://twemoji.maxcdn.com/2/'.
twemojiExtension String '.png', '.svg', '.jpg', '.jpeg', '.ico' This is a variable to define emoji arts static resources extension. Note: This property only changes the extension, the filename will always be the codepoint for the emoji Unicode. Default: '.png'.
twemojiFolder String Any String folder name This is a variable to define the folder where are stored the emoji arts static resources in the 'twemojiPath'. Default: '72x72'.
outerDivClasses Array of Strings Array of Strings (CSS classes) This is an array of CSS classes for those who wish to further customize the style of the outer div of the component (which encompasses emoji picker and textarea). Default: [].
btnEmojiClasses Array of Strings Array of Strings (CSS classes) This is an array of CSS classes for those who wish to further customize the style of the emoji button of the component. Default: [].
cooltextAreaClasses Array of Strings Array of Strings (CSS classes) This is an array of CSS classes for those who wish to further customize the style of the textarea of the component. Default: [].
Component Events
Event Parameter Description
contentChanged None This event fires everytime the textarea content is updated.
enterKey None This event fires everytime the enter key is pressed or the send button is clicked, so that you can define what action is to be performed.
emojiUnicodeAdded emojiUnicode This event fires everytime the user selects an emoji in the picker, and its parameter is the emoji in its Unicode format.
emojiImgAdded emojiImgTag This event fires everytime the user selects an emoji in the picker, and its parameter is the emoji HTML IMG tag.
Component Useful Methods
These are useful methods of the component you can access if you define a ref attribute to your component. The method can then be accessed on your Vue parent component with:
this.$refs.yourRefName.methodName
Method Parameter Description
addText(text) text - String This method adds a text to the textarea. The text parameter will then be sanitized and emojis Unicodes converted to emoji IMG arts from Twemoji.
addTextBlur(text) text - String This method adds a text to the textarea. The text parameter will then be sanitized and emojis Unicodes converted to emoji IMG arts from Twemoji. It is the same as addText(text) method, but this one doesn't trigger smartphone keyboard.
cleanText() None This method cleans the textarea.
setSkinTone(tone) tone - Number (a number >= 0 and <= 5) This method sets the skin tone active in the emoji picker.
Emoji Data Index
As seen in the example you will need, for the emoji picker to work, to import an indexed list of emojis and emoji groups name (which is provided by this lib) and pass as a prop of the CoolTextArea component. There is already different bundles so you are able to choose if you want your emoji picker to have all emojis or only smileys, or only smileys and objects, etc.
This library execute some optimizations excluding unnecessary fields (for this component) in an already indexed list of Unicode emojis from the lib 'emojibase'. Below you have info for each emoji bundle:

'cooltextarea/src/emoji-data/emoji-group-flags.json' 14KB 2KB
'cooltextarea/src/emoji-data/emoji-group-food-drink.json' 7KB 2KB

PS: You can provide your own emoji.json index file with a list of emoji Unicodes, but you will need to follow the structure of the bundles generated by this library (like cooltextarea/src/emoji-data/emoji-all-groups.json).
PS: You can provide your own emoji-groups.json index file with a list of emoji groups name, but you will need to follow the structure of the bundle generated by this library (cooltextarea/src/emoji-data/emoji-groups.json).
